## probrem

## 難しさ
簡単
## 解法
まずはファイルの種類を検索したいのでターミナルでubuntuを立ち上げたる。<br>
まずはターミナルでｗｓｌと打つ。<br>
ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー<br>
ｗｓｌ<br>
ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー<br>
そしてファイルの種類検索でascll　textと出る。<br>
これはascllコード表で16進数と文字を対応させている。<br>
なのでファイルの中身(16進数)を文字化するコマンドを使用。<br>
ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー<br>
strings flag<br>
ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー<br>
するとフラグが出現。
## 使用コマンド
### linux立ち上げ
wsl
### ファイルの種類検索
strings ファイル名
