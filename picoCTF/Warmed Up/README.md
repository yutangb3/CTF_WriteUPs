## probrem
What is 0x3D (base 16) in decimal (base 10)?
## 難しさ
簡単
## 解法
baseは進数、decimalは10進数です。<br>
16進数で0x3Dを10進数に変換します。<br>
3D=**16**×3+**1**×13=61

## 答え

## ここから学んだこと

0はこれは文字じゃなくて数字ですよというサイン。<br>
xはhe**x**adecimalが由来ということ。<br>
つまり使う時には0x(数字)
