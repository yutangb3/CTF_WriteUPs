## probrem
Can you convert the number 42 (base 10) to binary (base 2)?
## 難しさ
簡単
## 解法
  42をに二進数に直すだけです。<br>
  42=32×**1**+16×**0**+8×**1**+4×**0**+2×**1**+1×**0**
  つまり101010(2)
## 使用コマンド
## 答え
```picoCTF{101010}```
